# Moon Drifter Survival

A small browser game about an astronaut surviving on the moon in low gravity.

## Play

Open [index.html](/Users/juliantrujillo/Documents/New project/index.html) in a browser.

## Controls

- `A` / `D`: move
- `W`: low-gravity jump
- `E`: collect nearby supplies
- `R`: eat stored food

## Goal

Gather `food`, `ore`, `ice`, and `parts`, then build enough structures to keep the astronaut alive and stabilize the moon base.
